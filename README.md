# Oldagram

A mobile-first Instagram clone that brings the world of 19th-century Post-Impressionist art into the modern social media era. This project focuses on responsive design and dynamic DOM rendering using vanilla JavaScript.

## 🖼️ Demo
[Project Screenshots/Video](https://drive.google.com/drive/folders/1q6mR1s9LsDpd-PQUYIjFBRS7b30-uEH5?usp=sharing)

[Project link demo](https://assadullahhassan.github.io/oldagram/)


## 🚀 Features
* **Dynamic Rendering**: Posts are generated dynamically from an array of objects using JavaScript.
* **Interactive Likes**: Click the "heart" icon or double-tap the image to increase the like count
* **Fully Responsive**: Optimized for mobile, tablet, and desktop views.
* **Thematic Design**: Recreates the Instagram UI with a historical twist, featuring Vincent van Gogh and other iconic figures.


## 📂 Project Structure
     ├── index.html      # Main entry point
     ├── index.css       # Styling and layouts
     ├── index.js        # Data array and rendering logic
     └── images/         # Local assets (avatars, post images, icons)
